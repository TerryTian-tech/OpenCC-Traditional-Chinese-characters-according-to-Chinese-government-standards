# Change History of 规范繁体字形转换器

## Version 1.3.10

2026年7月8日

* **Fixed**：
    * 修正了繁体转换为简体模式遗漏13组异体转换关系的问题，这些异体字转换关系已在《〈通用规范汉字表〉表外正字的转换》中声明采纳
    * 跟随OpenCC官方词库修正复/覆转换错误的问题

## Version 1.3.9

2026年7月4日

* **Improved**：
    * 调整异体字转换关系，删去32组非《通规》正字转换关系
    * OpenCC转换引擎更新至1.4.0版本

* **Fixed**：
    * 修正了繁体转换为简体时，“坏”被固定转换为“坯”的问题

* **Note**：
    * 此版本及之后的Linux Deb包只针对Deepin和GXDE构建，不再在Ubuntu/Linux Mint上测试兼容性

## Version 1.3.8

2026年7月2日

* **Improved**：
    * 调整及新增异体字转换关系，调整后异体字较上一版本新增378组。进一步规范转换行为，非《通规》正字原则上不再转换
    * OpenCC转换引擎更新至1.3.2版本
    * Certifi库更新到2026.6.17版本
    * Beautifulsoup4库更新到4.15.0版本
    * Pywin32库更新到312版本。
    * Windows7兼容版本逐步迁移到OpenCC的Python替代版本，此版本为首个迁移后的兼容版本

## Version 1.3.7

2026年6月7日

* **New**：
    * 转换前允许选择一个文件夹内的部分文件进行批量转换，而不必转换一个文件夹内的所有文件

* **Fixed**：
    * 词库跟随OpenCC官方修正转换错误若干，更新到官方仓库2026.6.6的版本
    * 根据《辞海》第7版更新取消“紬—绸”异体字转换关系

## Version 1.3.6

2026年5月31日

* **New**：
    * 增加EPUB文件转换支持

* **Fixed**：
    * 词库跟随OpenCC修正转换错误若干

## Version 1.3.5

2026年5月10日

* **Improved**：
    * OpenCC转换引擎更新至1.3.1版本
    * Certifi库更新到2026.4.22版本

* **Fixed**：
    * 繁体转规范繁体模式取消对“於—于”的过度纠正

## Version 1.3.4
